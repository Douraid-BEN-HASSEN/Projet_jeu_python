Classe CPersonnage

Classe correspondant au personnage, elle peut permettre au joueur de se deplacer.

Module : from CPersonnage import CPersonnage
Classe mère : /
Classe(s) fille(s) : /

Documentation des membres

CPersonnage : map

Correspond à l'espace de jeu parent CMap.

CPersonnage : color

Correspond à la couleur du personnage.
Exemple : "#FFFFFF" ou "white"
Valeur par defaut : "#FFFFFF".

CPersonnage : width

Correspond à la largeur du personnage.
Valeur par défaut : 20.

CPersonnage : height

Correspond à la hauteur du personnage.
Valeur par défaut : 20.

CPersonnage : x

Correspond à la position x du personnage par rapport au cadre parent map.

CPersonnage : y

Correspond à la position y du personnage par rapport au cadre parent map.

CPersonnage : personnage

Correspond au corps du personnage sous forme de canvas rectangle qui a pour parent map.

CPersonnage : speed

Correspond à la vitesse du personnage en ms
Exemple : 7 (moyen)
Valeur par défaut : 10 (facile).

CPersonnage : avancement

Correspond au coefficient d'avancement du personnage.
Exemple : 5 (le personnage se déplace de 5px en 5px)
Valeur par défaut : 1 (le personnage se déplace de 1px en 1px).

CPersonnage : game_running

Variable correspondant à l'état du jeu : True le jeu tourne, False le jeu n'est pas en marche. elle permet entre autre de gérer les déplacement du joueur.

CPersonnage : startPos[2]

Tableau d'entiers contenant la position de départ X et Y du personnage.
Valeur par défaut : [-1,-1]

CPersonnage : movementTab[]

Tableau d'entiers contenant l'historique des touches pressées par l'utilisateur.

Documentation des fonctions

CPersonnage : CPersonnage(pmap : CMap, pcolor="#FFFFFF" : string, pwidth=20 : int, pheight=20 : int, px=0 : int, py=0 : int, pstarpos=[-1,-1] : int[2])

Construit un CPersonnage avec les paramètres suivants :
pmap : zone de jeu parent CMap.
pcolor : couleur du personnage. Valeur par défaut : "#FFFFFF".
pwidth : largeur du personnage. Valeur par défaut : 20.
pheight : hauteur du personnage. Valeur par défaut : 20.
px : position x du personnage par rapport au cadre map.canvas. Valeur par défaut : 0.
py : position y du personnage par rapport au cadre map.canvas. Valeur par défaut : 0.
pstarpos : position de départ du personnage. Valeur par défaut : [-1, -1].

CPersonnage : obj_init()

Fonction qui initialise l'objet CPersonnage :
elle crée le corps du personnage.
elle affiche le personnage dans la zone de jeu.
elle fait le lien entre les evenements et les fonctions de deplacement.

CPersonnage : movement()

Fonction qui deplace le CPersonnage a ses coordonnees x et y

CPersonnage : left()

Fonction qui est appelée quand Left est pressée :
Elle décrémente x à hauteur de x-avancement.

CPersonnage : right()

Fonction qui est appelée quand Right est pressée :
Elle incrémente x à hauteur de x+avancement.

CPersonnage : up()

Fonction qui est appelée quand Up est pressée :
Elle incrémente y à hauteur de y+avancement.

CPersonnage : keyup(e : event <KeyRelease>)

Fonction qui est appelée quand l'une des touches Haut, Bas, Gauche ou Droite est relachée :
Elle retire la touche relachée du tableau movementTab.

CPersonnage : keydown(e : event <KeyPress>)

Fonction qui est appelée quand l'une des touches Haut, Bas, Gauche ou Droite est pressée :
Elle ajoute la touche pressée dans le tableau movementTab.
