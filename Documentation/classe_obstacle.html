Classe CObstacle

Cette classe représente la classe mère des objets obstacles, elle correspond à un obstacle dans la zone de jeu qui fait perdre l'utilisateur : si un objet de type CPersonnage est en contact avec.

Module : from CObstacle import CObstacle
Classe mère : /
Classe(s) fille(s) : /

Membres

+ CMap map
+ string color
+ int width
+ int height
+ int x
+ int y
+ tkinter.Canvas rectangle obstacle
+ int speed

Fonctions publiques

Constructeur : CObstacle(pmap : CMap, pcolor="#696969" : string, pwidth=100 : int, pheight=20 : int, px=0 : int, py=0 : int)
+ obj_init()
- movement()
+ move(px : int, py : int)

Documentation des membres

CObstacle : map

Correspond à l'espace de jeu parent CMap.

CObstacle : color

Correspond à la couleur de l'obstacle.
Exemple : "#696969" ou "white"
Valeur par defaut : "#696969".

CObstacle : width

Correspond à la largeur de l'obstacle.
Valeur par défaut : 100.

CObstacle : height

Correspond à la hauteur de l'obstacle.
Valeur par défaut : 20.

CObstacle : x

Correspond à la position x de l'obstacle par rapport au cadre parent map.

CObstacle : y

Correspond à la position y de l'obstacle par rapport au cadre parent map.

CObstacle : obstacle

Correspond au corps de l'obstacle sous forme de canvas rectangle qui a pour parent map.

CObstacle : speed

Correspond à la vitesse de déplacement l'obstacle en ms
Valeur par défaut : 1.

Documentation des fonctions

CObstacle : CObstacle(pmap : CMap, pcolor="#696969" : string, pwidth=20 : int, pheight=20 : int, px=0 : int, py=0)

Construit un CObstacle avec les paramètres suivants :
pmap : zone de jeu parent CMap.
pcolor : couleur de l'obstacle. Valeur par défaut : "#696969".
pwidth : largeur de l'obstacle. Valeur par défaut : 100.
pheight : hauteur de l'obstacle. Valeur par défaut : 20.
px : position x de l'obstacle par rapport au cadre map.canvas. Valeur par défaut : 0.
py : position y de l'obstacle par rapport au cadre map.canvas. Valeur par défaut : 0.

CObstacle : obj_init()

Fonction qui initialise l'objet CObstacle :
elle crée le corps de l'obstacle.
elle affiche l'obstacle dans la zone de jeu.

CObstacle : movement()

Fonction qui deplace le CObstacle a ses coordonnees x et y

CObstacle : move(px : int, py : int)

Fonction qui change les coordonnees x et y
