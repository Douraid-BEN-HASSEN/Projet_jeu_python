CSetting : save_game(pdata : string) : bool

Fonction qui enregistre une sauvegarde, elle transforme un tableau d'objet (CMap, CArrivee, CPersonnage, CObstacle[], info_joueur) en texte de sauvegarde
- pdata le tableau d'état sous forme [CMap, CArrivee, CPersonnage, CObstacle[], info_joueur[fichier_path, joueur_nom, score, sauvegarde, niveau]].
Valeur de retour : True si tout est OK, False si il y a une erreur.
