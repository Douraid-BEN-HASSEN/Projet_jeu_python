Classe CMap

Classe correspondant au plateau du jeu, elle contient tous les elements (personnage, obstacles, etc...) elle peut ajouter ou supprimer des elements

Documentation des membres

CMap : width

Correspond à la largeur de la zone de jeu.
Valeur par défaut : 800.

CMap : height

Correspond à la hauteur de la zone de jeu.
Valeur par défaut : 600.

CMap : color

Correspond à la couleur de la zone de jeu.
Exemple : "#DCDCDC" ou "white"
Valeur par defaut : "#DCDCDC".

CMap : element

Tableau qui contient tous les élément de la zone de jeu.

CMap : fenetre

Correspond à la fenêtre principale.

CMap : canvas

Cadre de type tkinter.Canvas qui correspond au corps de la zone de jeu.

Documentation des fonctions

CMap : CMap(pwidth=800 : int, pheight=600 : int, pcolor="#DCDCDC" : string)

Construit un CMap avec les paramètres suivants :
pmap : zone de jeu parent CMap.
pcolor : couleur de l'arrivee. Valeur par défaut : "#DCDCDC".
pwidth : largeur de l'arrivee. Valeur par défaut : 800.
pheight : hauteur de l'arrivee. Valeur par défaut : 600.
