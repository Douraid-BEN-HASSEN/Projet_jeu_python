Classe CArrivee

Cette classe représente la classe mère des objets arrivées, elle correspond à l'emplacement d'arrivée dans la zone de jeu : si un objet de type CPersonnage est en contact avec.

Module : from CArrivee import CArrivee
Classe mère : /
Classe(s) fille(s) : /

Membres

+ CMap map
+ string color
+ int width
+ int height
+ int x
+ int y
+ tkinter.Canvas rectangle arrivee
+ int speed

Fonctions publiques

Constructeur : CArrivee(pmap : CMap, pcolor="#00FF00" : string, pwidth=50 : int, pheight=50 : int, px=0 : int, py=0 : int)
+ obj_init()
- movement()
+ move(px : int, py : int)

Documentation des membres

CArrivee : map

Correspond à l'espace de jeu parent CMap.

CArrivee : color

Correspond à la couleur de l'arrivee.
Exemple : "#00FF00" ou "white"
Valeur par defaut : "#00FF00".

CArrivee : width

Correspond à la largeur de l'arrivee.
Valeur par défaut : 50.

CArrivee : height

Correspond à la hauteur de l'arrivee.
Valeur par défaut : 50.

CArrivee : x

Correspond à la position x de l'arrivee par rapport au cadre parent map.

CArrivee : y

Correspond à la position y de l'arrivee par rapport au cadre parent map.

CArrivee : arrivee

Correspond au corps de l'arrivee sous forme de canvas rectangle qui a pour parent map.

CArrivee : speed

Correspond à la vitesse de déplacement de l'arrivee en ms
Valeur par défaut : 1.

Documentation des fonctions

CArrivee : CArrivee(pmap : CMap, pcolor="#00FF00" : string, pwidth=50 : int, pheight=50 : int, px=0 : int, py=0)

Construit un CArrivee avec les paramètres suivants :
pmap : zone de jeu parent CMap.
pcolor : couleur de l'arrivee. Valeur par défaut : "#00FF00".
pwidth : largeur de l'arrivee. Valeur par défaut : 50.
pheight : hauteur de l'arrivee. Valeur par défaut : 50.
px : position x de l'arrivee par rapport au cadre map.canvas. Valeur par défaut : 0.
py : position y de l'arrivee par rapport au cadre map.canvas. Valeur par défaut : 0.

CArrivee : obj_init()

Fonction qui initialise l'objet CArrivee :
elle crée le corps de l'arrivee.
elle affiche l'arrivee dans la zone de jeu.

CArrivee : movement()

Fonction qui deplace le CArrivee a ses coordonnees x et y

CArrivee : move(px : int, py : int)

Fonction qui change les coordonnees x et y
